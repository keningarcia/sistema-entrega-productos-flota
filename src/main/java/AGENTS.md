#Nombre del proyecto
Solución para una flota de unidades de camiones (Check list, recepción y entregas de productos a clientes)

#Requerimientos
Login
Estado de la unidad
Inicio de ruta (Indicar el kilometraje incial)
Lista de clientes
En cada cliente indicar el pedido solicitado y opción de entrego o cancelado
Termino de ruta (Indicar el kilometraje final)

#Estructura del proyecto
-controller
-dto
-entity
-repository
-service

todo debe de estar en cada carpeta segun su función
